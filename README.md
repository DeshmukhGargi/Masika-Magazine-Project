# Masika Magazin

Masika Magazin is my **first Web Technologies (WT) project**, developed during my **first year of study**.  
This project marks my introduction to web development and version control using Git and GitHub.

## Project Description
Masika Magazin is a simple **online magazine website** created to understand the basics of web design and development.  
The main goal of this project is to practice structuring web pages and presenting content in a clean and readable format.

## Features
- Simple and clean magazine-style layout
- Basic responsive design
- Organized content sections
- Beginner-friendly implementation

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub
